• $L (1 \leq L)$ is max number on a lottery game. ex. $L$ = 31
  • $l (1 \leq l \leq L)$ is pick number on a lottery game. ex. $l$ = 5
  • $n (1 \leq n)$ is current number of times. ex. $n$ = 1115
  • $\mathbf{A}_n$ is array of flag on the number with the size of $L$. ex. $\mathbf{A}_1$ = [0, 1, 0, ..., 0, 1, 0]
  • $\mathbf{B}_n = \Sigma_{i=1}^n \mathbf{A}_i$ is accumulatation. ex. $\mathbf{B}_3$ = [0, 1, 2, ..., 0, 3, 1]
  • $\mathbf{R}_n = \frac{ \mathbf{B}_n }{ n \cdot l }$ is ratio of each number, so that $|\mathbf{R}_n|=1$. ex. $\mathbf{R}_3$ = [0, 1/15, 2/15, ..., 0, 3/15, 1/15]
  • $\mathbf{P}_{n+1} = \frac{ 1 - \mathbf{R}_n }{ L - 1 }$ is probability of each number on next time. ex. $\mathbf{P}_4$ = [1/30, 14/450, 13/450, ..., 1/30, 12/450, 14/450]
