Variables

  • $L (1 \leq L)$ is max number on a lottery game. ex. $L$ = 31
  • $p (1 \leq p \leq L)$ is pick number on a lottery game. ex. $p$ = 5
  • $N (1 \leq N)$ is current number of times. ex. $N$ = 1115
  • $\mathbf{A}_n$ is array of pick numbers in all games. ex. $\mathbf{A}_1$ = [1, 5, 9, 17, 22]
  • $\mathbf{B}_n$ is array of pick numbers in binary with the size of $L$. ex. $\mathbf{B}_1$ = [0, 1, 0, ..., 0, 1, 0]
  • $\mathbf{C}_n = \Sigma_{i=1}^n \mathbf{B}_i$ is accumulatation. ex. $\mathbf{C}_3$ = [0, 1, 2, ..., 0, 3, 1]
  • $\mathbf{R}_n = \frac{ \mathbf{B}_n }{ p \cdot n }$ is ratio of each number, so that $|\mathbf{R}_n|=1$. ex. $\mathbf{R}_3$ = [0, 1/15, 2/15, ..., 0, 3/15, 1/15]
  • $\mathbf{P}_{n+1} = \frac{ 1 - \mathbf{R}_n }{ L - 1 }$ is probability of each number on next time. ex. $\mathbf{P}_4$ = [1/30, 14/450, 13/450, ..., 1/30, 12/450, 14/450]
